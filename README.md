# ✦ 非人类测试 ✦

一个有趣的网页测试 —— 长按按钮，发现你内心深处的真实灵魂！

## 玩法

1. 按住屏幕中央的按钮
2. 外圈进度条会逐渐充满
3. **中途松手则进度归零**，需要重新按压
4. 进度读满后揭晓你的「真身」

## 特色

- **30 种动物结果**，分为 4 个稀有度等级
- **设备锁定**：同一设备测试结果固定不变（基于设备指纹生成）
- **稀有动物**：独角兽、凤凰、龙等传说级动物出现概率仅约 3%
- **全平台适配**：手机 / 平板 / 桌面端均可使用
- **分享功能**：支持原生分享与剪贴板复制

## 动物稀有度

| 等级 | 概率 | 数量 |
|------|------|------|
| 普通 | ~53% | 12 种 |
| 稀有 | ~30% | 9 种 |
| 罕见 | ~13% | 6 种 |
| 传说 | ~3%  | 3 种（独角兽、龙、凤凰）|

## 部署

这是一个纯静态页面，直接打开即可运行：

```bash
# 或者用任何 HTTP 服务器
npx serve .
```

## 技术栈

- 原生 HTML / CSS / JavaScript（零依赖）
- SVG 环形进度条
- Canvas 星空背景 + 彩纸特效
- 设备指纹哈希 + 加权随机算法
